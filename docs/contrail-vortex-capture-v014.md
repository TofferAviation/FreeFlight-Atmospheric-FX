# Contrail vortex capture calibration

The next visual pass changes the young side-vortex particles from an expanding puff chain into a compact capture layer:

- High-rate, small wingtip particles for a continuous wisp rather than dotted beads.
- Particle scale contracts over lifetime as the visible side vortex is pulled into the wake.
- Primary wake particles briefly develop, then contract and fade instead of growing into large blobs.
- Main engine contrail particles remain dense but stop expanding to oversized late-life discs.

This is a visual approximation using X-Plane particle lifetime curves. It does not claim to be a CFD simulation.
