FFAtmospherics v0.4 standalone development package

Required installed structure:
  FFAtmo\FFAtmoCompanion.exe
  FFAtmo\64\win.xpl
  FFAtmo\assets\FFAtmo_Lineage.obj
  FFAtmo\assets\FFAtmo_particles.pss
  FFAtmo\assets\FFAtmo_particles.png
  FFAtmo\profiles\Lineage1000.ini

Copy the complete FFAtmo folder into X-Plane 12\Resources\plugins.
Launch FFAtmoCompanion.exe for the six-page live control application.

The companion is optional during flight; win.xpl continues running with the
last saved settings when the app is closed.
